Proof of Concept (POC) - Plateforme de Sécurité des Données de Santé

Ce Proof of Concept (POC) vise à démontrer la pertinence et l'efficacité d'une solution intégrée pour la sécurisation des données de santé dans le domaine médical, en conformité avec la Politique Générale de Sécurité des Systèmes d'Information de Santé (PGSSI-S).

## Présentation du Projet

La sécurisation des données de santé est une préoccupation majeure dans le secteur médical. Ce POC propose une plateforme sécurisée pour la gestion des données médicales, offrant les fonctionnalités suivantes :
- Inscription et connexion des professionnels de santé
- Gestion des patients : ajout, suppression et consultation des profils des patients
- Génération de QR codes uniques pour chaque patient
- Visualisation et modification des profils des patients

## Technologie Utilisée

- Backend : Flask (Python)
- Frontend : HTML, CSS, JavaScript
- Base de données : SQLite
- Autres technologies : Flask SQLAlchemy, Flask Login, qrcode

## Mise en Oeuvre

1. Installation des dépendances :
   ```
   pip install -r requirements.txt
   ```

2. Configuration de la base de données :
   - Modifier la variable `SQLALCHEMY_DATABASE_URI` dans `config.py` selon les besoins


3. Lancement de l'application :
   ```
   python run.py
   ```
